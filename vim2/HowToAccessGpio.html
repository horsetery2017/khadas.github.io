How To Access GPIO

Here is about how to access GPIO on Android and Ubuntu.

Preconditions

The Rom must satisfy the following conditions:

  • Android M >= V170603
  • Android N >= V170421
  • Ubuntu Server >= V170515
  • Ubuntu Mate >= V170605

On Android

GPIO List

  • ADB Command
