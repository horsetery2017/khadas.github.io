Create Burning Card Via Command Line

This is a step by step instructions to create burning card for linux users, you can also use the Windows approach instead.

Preperations:

  • Build to get or download the latest U-Boot file for SD card.
  • You might need to format the TF Card Via fdisk if your card exists more than one partitions.
