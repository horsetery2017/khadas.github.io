Banks:

Pins:

For example, get the number of GPIOAO_6:
Number(GPIOAO_6) = bank + pin = 0 - 0 + 6 = 6

periphs-banks:
Banks:

Pins:

For example, get the number of GPIOH_5:
Number(GPIOH_5) = bank + pin = 11 - 11 + 32 = 32
